# Rock RMS UI & Experience Toolkit

A collection of reusable Rock RMS components, code snippets, Lava templates, HTML blocks, CSS enhancements, JavaScript utilities, and dashboard patterns designed to improve user experience, increase visibility into ministry data, and unlock additional capabilities within Rock RMS.

## Purpose

Rock RMS is incredibly powerful, but many organizations only scratch the surface of what's possible.

This repository exists to provide practical, copy-paste-ready solutions that help Rock administrators, developers, and ministry leaders:

* Build more engaging user experiences
* Surface meaningful ministry data
* Improve dashboard usability
* Modernize Rock interfaces
* Accelerate adoption of Rock RMS features
* Create custom ministry workflows
* Extend Rock without building plugins

## Philosophy

The goal of this repository is to:

### Prioritize Ministry

Technology should support ministry, not create complexity.

### Improve Adoption

People engage more effectively when tools are intuitive and enjoyable to use.

### Reduce Development Overhead

Many powerful experiences can be built using:

* Lava
* HTML
* CSS
* JavaScript

without creating custom plugins.

### Stay Upgrade Friendly

Whenever possible, solutions are designed to:

* Avoid modifying core Rock files
* Minimize upgrade risks
* Leverage supported Rock functionality
