
# Vertical Image Marquee

A lightweight vertical scrolling image marquee component for Rock RMS pages, dashboards, and landing pages.

## Features

- Vertical infinite scroll
- Randomized image order on each page load
- Seamless duplicated image loop
- Rounded image corners
- Responsive full-width images
- Placeholder asset paths for public repo use
- No external dependencies

## How It Works

The component uses JavaScript to populate the marquee from an image array.

  "/assets/marquee-image-01.jpg",
